## How To Use

1. 메인 화면 중앙의 '테스트 시작' 버튼을 통해 테스트를 시작할 수 있습니다.
2. 테스트가 끝난 후 결과 화면의 'COPY URL' 버튼을 통해 링크를 공유할 수 있습니다.
3. 게시글 작성 페이지에서 본인의 테스트 결과와 취미 활동을 사람들과 공유할 수 있습니다.

## How To Install

1. Flask 설치

```
pip install flask
```

2. 'main.py' 실행
